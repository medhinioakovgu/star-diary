# Mobile – React Native (Expo) Frontend

This directory contains the **React Native / Expo** mobile frontend for The Star-Interview Diary.

## Quick-start

### 1. Initialise the Expo project (run once)

```bash
npx create-expo-app mobile -t expo-template-blank-typescript
```

> This command scaffolds a blank TypeScript Expo project inside the `mobile/`
> directory. It installs all dependencies (React Native, Expo SDK, etc.) via npm.

### 2. Copy the custom screen

After initialising, copy (or move) `ChatScreen.tsx` into the project:

```bash
# If you ran create-expo-app from the repo root the mobile/ folder already
# exists; just make sure ChatScreen.tsx is at mobile/ChatScreen.tsx.
cp ChatScreen.tsx mobile/ChatScreen.tsx   # if needed
```

### 3. Register the screen in your navigator

Open `mobile/App.tsx` (generated by Expo) and replace its contents with:

```tsx
import { SafeAreaView, StatusBar, StyleSheet } from 'react-native';
import ChatScreen from './ChatScreen';

export default function App() {
  return (
    <SafeAreaView style={styles.container}>
      <StatusBar barStyle="dark-content" />
      <ChatScreen />
    </SafeAreaView>
  );
}

const styles = StyleSheet.create({
  container: { flex: 1 },
});
```

### 4. Start the Expo dev server

```bash
cd mobile
npx expo start
```

Press **`i`** to open in the iOS Simulator, **`a`** for the Android Emulator, or
scan the QR code with the **Expo Go** app on a physical device.

## Backend connectivity

| Platform | API base URL |
|---|---|
| iOS Simulator | `http://localhost:8000` |
| Android Emulator | `http://10.0.2.2:8000` |
| Physical device | `http://<your-machine-LAN-IP>:8000` |

`ChatScreen.tsx` automatically selects `localhost` vs `10.0.2.2` based on
`Platform.OS`. For physical devices, update `API_BASE` in `ChatScreen.tsx` to
your machine's LAN IP (e.g. `http://192.168.1.100:8000`).

```
mobile/
├── assets/            # App icons and splash images (generated)
├── node_modules/      # Dependencies (git-ignored)
├── App.tsx            # Root component (register ChatScreen here)
├── ChatScreen.tsx     # ⭐ AI chat UI (this file)
├── app.json           # Expo configuration
├── babel.config.js    # Babel configuration
├── package.json       # npm dependencies
```
